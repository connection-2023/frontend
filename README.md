### ✅commit 규칙

|   Type   | 내용                                              |
| :------: | ------------------------------------------------- |
|   feat   | 새로운 기능 추가                                  |
|   fix    | 버그 수정 또는 typo                               |
| refactor | 리팩토링                                          |
|  style   | 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우 |

<br />

1. Issues 등록: `Title : [이슈 관련 태그] 이슈 내용`
2. 커밋 메세지: `Issues 관련 태그(#Issues): 내용` <br/> 예시) `feat(#27): 게시판 기능 구현`
